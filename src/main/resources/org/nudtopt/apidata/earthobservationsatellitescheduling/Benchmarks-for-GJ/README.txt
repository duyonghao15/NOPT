The 3D rotation angle Δw (in radian) can be calculated by the following equation：

Δw = 2 * arccos { 0.5 * [1 + sin(Δθ)*sin(ΔΦ)*sin(Δψ) + cos(Δθ)*cos(ΔΦ) + cos(Δθ)*cos(Δψ) + cos(ΔΦ)*cos(Δψ) ] ^ 0.5 }

where Δθ, ΔΦ and Δψ determine the (positive) rotation angles of pitching, rolling and yawing, respectively.
